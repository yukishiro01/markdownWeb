howdown.js

强大的 JavaScript Markdown 解析器，支持 GitHub Flavored Markdown 语法。

核心特性

  • 实时预览转换
  • 扩展的 Markdown 语法
  • 自定义渲染规则 
const converter = new showdown.Converter({
    tables: true,
    tasklists: true,
    strikethrough: true,
    emoji: true,
    extensions: [{
        type: 'lang',
        regex: /={2}([^=]+)={2}/g,
        replace: '$1'
    }]
});

💡

项目起源

MarkdownWeb 项目源于对现代化 Markdown 编辑体验的追求。在当今的文档编辑领域， 我们发现许多优秀的编辑器要么依赖后端服务，要么需要安装客户端。这促使我们思考： 能否创建一个纯前端的、即开即用的 Markdown 编辑器？

纯前端实现 即开即用 无需安装
⚙️

技术选型

通过整合 Showdown.js 的强大解析能力和 TailwindCSS 的现代化设计， 我们打造了这个轻量级但功能完整的编辑器。它不仅满足了基本的文档编辑需求， 还通过本地存储实现了数据持久化。

轻量级 功能完整 本地存储
🎯

设计理念

简单但不简陋

追求简洁的界面设计，但不牺牲功能的完整性。让用户可以专注于写作本身。

用户体验优先

从用户的角度出发，提供流畅的编辑体验和直观的操作方式。

持续改进

保持开源开放的态度，不断收集用户反馈，持续优化和改进产品。
